# Gerenciador de vendas
Um projeto para gerenciar produtos e vendas

# Descrição
Este projeto foi criado para consolidar meus conhecimentos utilizando o framework .NET, para ser mais especifico a plataforma de desenvolvimento web ASP.NET Core ele imita um pequeno modulo
de um sistema para o gerenciamento de vendas, com controle de estoque, autorização para acesso da aplicação utilizando a biblioteca do Identity, tudo isso utilizando um banco de dados em memória
SQLite para facil utilização em qualquer maquina.

# Instalação
Caso não tenha o dotnet instalado será necessário instalar através do site:
https://dotnet.microsoft.com/pt-br/download

Após a instalação do dotnet, clone o repósitorio através do seguinte comando no terminal:
git clone https://github.com/Drebirth/GerenciadorDeVendas.git

# Uso
Para iniciar através da linha de comando:
dotnet run
